## Option 1 – Add workflow scope to your PAT
1. Go to https://github.com/settings/tokens
2. Edit your token, add the  scope
3. Run: git push origin main (from the .github/workflows folder)

## Option 2 – Create the workflow manually via GitHub.com
1. Go to https://github.com/lowserzeditexe-lab/blackwing-launcher
2. Click "Actions" → "New workflow" → "set up a workflow yourself"
3. Paste the content from  in this repo
